## Para rodar a aplicação 🖥️
- Instalação do node modules **(Front e Back)**:
  - `npm install`
- Configuração do SQLite **(Back)**:
  - `npm run migration:generate`
  - `npm run migration:run`
- Rodar a aplicação **(Front e Back)**:
  - `npm run start`
